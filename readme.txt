== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the BlueLevel 3D CSSProgress Bar button on tinyMCE editor to add a new BlueLevel 3D CSSProgress Bar progress bar.
4. Clicking on the BlueLevel 3D CSSProgress Bar button a new modal will open.
5. Choose the title, color, style and value (upto 100) of your BlueLevel 3D CSSProgress Bar. Click ok, and it will add a shortCode.
6. You're Done!!!
